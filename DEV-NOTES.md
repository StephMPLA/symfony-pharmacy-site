# Notes de développement – Tailwind / AssetMapper

## Contexte
Ce projet utilise :
- Symfony 7.4
- AssetMapper
- Tailwind CSS via symfonycasts/tailwind-bundle
- Node.js **uniquement pour l’autocomplétion IDE (PhpStorm)**

⚠️ Node.js / npm ne sont **PAS utilisés en production**.  
Ils servent uniquement à l’autocomplétion Tailwind dans PhpStorm.
